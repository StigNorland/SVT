- **Path A (statistical):** removing the muon (3/2) or pion (2) collapses μ₀'s

  spectrum is **not evenly spaced** (rungs 1.5, 2.5, 8.5, 32.5 — gaps 1, 6, 24);

  artifacts (ruler-density snapping); the pion never appears as an eigenfrequency.
- **#76 (Volovik Berry phase):** scalar BdG holonomy is trivial (γ_B=0); no
  half-integer ladder.

  amplitude `|V|=0` exactly, so γ_B=0 — the half-integer ladder does not exist for
  the integer-vortex object either.

The verdict is consistent across five independent probes: **there is no derived

fights this evidence; #93 Task 3 is therefore resolved as a **demotion**, exactly
