# SSV I Supplemental Material

This directory collects Paper I support material for the SSV particle-as-vortex framework.

- `notes/` contains derivation notes copied from the working folder.
- `data/` contains small smoke-test outputs and summary tables.
- `../../instruments/paper_i/` contains the corresponding numerical prototypes.

Current status: these scripts support the Paper I program by constructing the LogSE vortex profile, toroidal background ansatz, mass-ladder estimates, and reduced BdG/muon-mode diagnostics. They are best read as exploratory derivation code rather than as final parameter-free simulations.
