# SSV III Outline

Working title: `Irreversible Time and Wake Entropy`

## Scope line

This paper addresses only the origin of irreversible time in SSV.

## Main thesis

Time is local change in `Psi`, the past survives only as wake structure in the present state, and entropy is the persistent wake-complexity of that present state.

## Core claims

1. Time is not a background dimension.
2. The past is not a place; it is encoded in present structure.
3. Events leave distributed wakes in `Psi`.
4. Physical time reversal is unavailable because no local process can globally unwrite the wake.
5. Entropy should be formulated as wake-complexity, not as a bare event counter.
