# SSV II Supplemental Numerics

This directory contains numerical support material for Paper II.

- `reconnection-barrier-results.md` summarizes the current 3D reconnection-barrier checks.
- `data/` contains the curated CSV tables referenced by the note.
- `../../src/paper_ii_reconnection_supplement.py` is the compact reproduction script.

Current status: the numerical work supports the structural claims that reconnection saddles exist, topology changes the amplitude/phase channel, and spectral chiral stiffness inflates cap geometry. It does not yet directly confirm the golden-ratio cap relation at physical scale.
