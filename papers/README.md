# Papers

This directory is for manuscript material, outlines, paper-specific notes,
result notes, and draft-support documents. Each `SSV-*` folder owns one focused
paper or focused supplement in the series.

## Folder Index

| Folder | Contents |
|---|---|
| `SSV-I/` | Paper I material for particles as topological defects: electron vortex scale, pion/proton assignments, static trefoil-breather numerics, muon/lepton no-go and spinor/HQV follow-ups. |
| `SSV-II/` | Paper II material for forces as hydrodynamic modes: gravity consistency checks, electroweak/reconnection geometry, order-parameter audits, and supporting numerical result notes. |
| `SSV-III/` | Complete draft of the time paper: irreversible time as local change in `Psi`, the past as wake structure, and entropy as persistent wake-complexity. |
| `SSV-IV/` | Complete draft of the gravity paper: gravity as motion through gradients in local update capacity, with weak-field bending, redshift, and pressure/refraction language. |
| `SSV-V/` | Complete draft of the black-hole paper: condensate black holes with frozen interiors, topological spin, no singularity, horizon memory, and stable remnant logic. |
| `SSV-VI-a/` | Galactic flat-curve paper: central black-hole standing-wave scale, disc soliton / Mestel profile, and focused rotation-curve phenomenology. |
| `SSV-VI-b/` | Galactic morphology companion: 2D disc resonance, azimuthal overtones, morphology sequence, wiggles, and why 1D rotation curves look smooth. |
| `SSV-VII-a/` | Quantum-mechanics paper: Madelung transformation, Schrodinger limit, quantisation as circulation topology, uncertainty, and measurement as reconnection. |
| `SSV-VII-b/` | Emergent-geometry paper: metric behaviour as a hydrodynamic/macroscopic limit and why quantizing gravity is mis-posed in SSV. |
| `SSV-VIII/` | Cosmogony paper: staged emergence from the permissive void through perturbation, interference, saturation, and defect nucleation. |
| `SSV-IX/` | Future CMB / primordial phonon-bath paper: scoped outline for CMB spectrum, primordial phonon bath, entropy-budget cosmology, and input from SSV VIII. |
| `SSV-Alpha/` | Focused alpha supplement: fine-structure constant framed as a toroidal-vortex aspect-ratio calculation, without claiming the full derivation is complete. |
| `SSV-Goldstone/` | Focused electromagnetic supplement: observed photon as the Goldstone/Bogoliubov phase mode at `c`, with chiral-shear assigned to static near-field topology. |
| `pdf/` | Human-readable compiled PDFs copied from the paper folders after successful builds. |

Keep one main claim per paper.
