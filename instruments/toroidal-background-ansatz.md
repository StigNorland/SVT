# Toroidal Background Ansatz for the Electron Vortex

This note builds an explicit background ansatz for the static toroidal defect
\(\Psi_0(\mathbf x)\) needed in the muon-mode programme.

## 1. Goal

We need a background field \(\Psi_0(\mathbf x)\) that does four jobs at once:

1. represents a singly quantized vortex ring,
2. has equilibrium major radius \(R_e = \xi/\alpha\),
3. reduces locally to the known straight-vortex core profile,
4. is simple enough to differentiate with respect to \(R\) for the breathing mode
   \(\Phi_R = R_e \partial_R \Psi_0\).

The point of this ansatz is not to claim the exact relaxed torus, but to provide a consistent
starting configuration for projection integrals and numerical experiments.

## 2. Geometry

Work in cylindrical coordinates \((r,\varphi,z)\), with the torus centered on the \(z\)-axis and
its ring lying in the plane \(z=0\).

Define the distance to the torus centerline
\[
s(r,z;R) \equiv \sqrt{(r-R)^2 + z^2}.
\]

This is the minor-radius coordinate: \(s=0\) is the vortex core ring.

Define also the local poloidal angle in the meridional \((r,z)\)-plane
\[
\vartheta(r,z;R) \equiv \operatorname{atan2}(z,\, r-R).
\]

The azimuthal angle around the major circle is the usual cylindrical angle \(\varphi\).

So the torus is parameterized by

- `s`: distance from the core ring,
- `vartheta`: angle around the minor cross-section,
- `varphi`: angle along the major ring.

## 3. Leading static ansatz

The simplest singly quantized toroidal vortex is obtained by lifting the straight-vortex core
profile onto the torus:
\[
\Psi_0(r,\varphi,z;R)
=
f_0\!\left(\frac{s(r,z;R)}{\xi}\right)\,
e^{i\vartheta(r,z;R)}.
\]

Here \(f_0(\tilde s)\) is the same monotone planar core profile used in Appendix C, with

\[
f_0(0)=0,\qquad
f_0(\infty)=1.
\]

This ansatz has the correct local winding: when one circles the core in the minor cross-section,
\(\vartheta\) changes by \(2\pi\), so the phase winds once and the circulation is quantized.

## 4. Why this is the correct zeroth-order torus

Near any small patch of the ring, the geometry is locally that of a straight vortex line. The
straight-vortex profile is therefore the correct inner solution for the core. Curvature enters
only through the slow variation of the embedding of the ring on scales of order \(R_e\), with
\[
\frac{\xi}{R_e}=\alpha \ll 1.
\]

So the ansatz
\[
\Psi_0 = f_0(s/\xi)e^{i\vartheta}
\]
is the natural leading-order background in a thin-core expansion.

## 5. First curvature correction

To go beyond leading order, allow the amplitude to depend weakly on position along the minor
angle:
\[
\Psi_0(r,\varphi,z;R)
=
\left[
f_0\!\left(\frac{s}{\xi}\right)
+\alpha\, f_1\!\left(\frac{s}{\xi}\right)\cos\vartheta
+O(\alpha^2)
\right]
e^{i\vartheta}.
\]

The \(\cos\vartheta\) term is the lowest toroidal asymmetry:

- it distinguishes the inner side of the torus \((r<R)\) from the outer side \((r>R)\),
- it is exactly the angular structure generated by curvature,
- it vanishes in the straight-vortex limit \(R\to\infty\).

So the minimal curved-torus ansatz is
\[
\Psi_0^{\rm curved}
=
\left[f_0(\tilde s)+\alpha f_1(\tilde s)\cos\vartheta\right]e^{i\vartheta},
\qquad
\tilde s=s/\xi.
\]

## 6. Major-radius breathing mode

The muon candidate couples to the major-radius breathing deformation, so we need the tangent
vector
\[
\Phi_R(\mathbf x)
\equiv
R_e \frac{\partial \Psi_0(\mathbf x;R)}{\partial R}\Big|_{R=R_e}.
\]

For the leading ansatz,
\[
\Psi_0 = f_0(s/\xi)e^{i\vartheta},
\]
the \(R\)-dependence enters only through \(s\) and \(\vartheta\). Using
\[
\frac{\partial s}{\partial R} = -\frac{r-R}{s} = -\cos\vartheta,
\qquad
\frac{\partial \vartheta}{\partial R} = \frac{z}{(r-R)^2+z^2} = \frac{\sin\vartheta}{s},
\]
we obtain
\[
\partial_R \Psi_0
=
e^{i\vartheta}
\left[
-\frac{\cos\vartheta}{\xi} f_0'(\tilde s)
+ i\,\frac{\sin\vartheta}{s} f_0(\tilde s)
\right].
\]

Therefore
\[
\Phi_R
=
R_e e^{i\vartheta}
\left[
-\frac{\cos\vartheta}{\xi} f_0'(\tilde s)
+ i\,\frac{\sin\vartheta}{s} f_0(\tilde s)
\right].
\]

This is the explicit breathing-mode seed for the projected BdG basis.

## 7. Chiral companion mode

The lowest transverse chiral mode should be orthogonal to pure breathing and localized near the
ring. The simplest seed ansatz is
\[
\Phi_\chi(r,\varphi,z)
=
i\,g_\chi(\tilde s)\,\cos\vartheta\, e^{i\vartheta},
\]
or, for the sine-parity partner,
\[
\Phi_\chi^{(\sin)}(r,\varphi,z)
=
i\,g_\chi(\tilde s)\,\sin\vartheta\, e^{i\vartheta}.
\]

The \(\cos\vartheta\) choice is the natural partner to curvature because it distinguishes inner
and outer sides of the torus, just as the first curvature correction does.

## 8. Boundary conditions

The background must satisfy:

- Core regularity:
  \[
  \Psi_0 \sim \tilde s\, e^{i\vartheta}
  \qquad \text{as } s\to 0.
  \]

- Vacuum saturation:
  \[
  |\Psi_0| \to 1
  \qquad \text{as } s\to\infty.
  \]

- Single-valuedness in physical space:
  the phase changes by \(2\pi\) only when a loop links the vortex ring once.

- Reflection symmetry:
  under \(z\to -z\), one has \(\vartheta\to -\vartheta\), so the modulus is even and the phase
  reverses as required by vortex orientation.

## 9. Static energy functional on the ansatz

With the background inserted into the static energy,
\[
E[\Psi_0]
=
\int d^3x\,
\left[
\frac{\hbar^2}{2m_0}|\nabla\Psi_0|^2
+V(\rho_0|\Psi_0|^2)
+\frac{\lambda\hbar^2}{2m_0\rho_0}(\nabla\times j_0)^2
\right],
\]
the equilibrium radius is obtained from
\[
\frac{dE(R)}{dR}\Big|_{R=R_e}=0.
\]

At leading order in \(\alpha=\xi/R_e\), this reproduces the radius condition already used in the
draft. At next order, the function \(f_1\) captures curvature corrections to the torus profile.

## 10. Minimal practical version

For immediate numerical work, use the simplest explicit background
\[
\boxed{
\Psi_0(r,\varphi,z;R_e)
=
f_0\!\left(\frac{\sqrt{(r-R_e)^2+z^2}}{\xi}\right)
\exp\!\Big(i\,\operatorname{atan2}(z,r-R_e)\Big)
}
\]
with \(f_0\) taken from the planar vortex solution.

Then use
\[
\Phi_R = R_e\,\partial_R \Psi_0\big|_{R=R_e}
\]
as the breathing seed, and add the first chiral seed
\[
\Phi_\chi = i\,g_\chi(\tilde s)\cos\vartheta\,e^{i\vartheta}.
\]

This gives a closed, differentiable starting point for:

1. projection integrals \(K_{ab}\),
2. BdG basis construction,
3. numerical relaxation of the curved torus.
